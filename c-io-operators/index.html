Study Questions

  1. (T or F) printf("%c\n", 'a'); will print the character a.
  2. (T or F) printf("%s\n", 'a'); will print the character a.
  3. (T or F) printf("%.*s", 5, "hello there\n"); prints out hello.

  4. (T or F) For printf(), the formatting argument G may generate the same results as the formatting argument f.

  5. The following program prints out hello there! when executed (T or F). 
     int main() {
   char array[20];
   sprintf(array, "%s", "hello there!");
 }
  6. (T or F) sscanf() reads characters from standard input.

  21. Study this program. Try commenting out the s[j] = '\0'; statement. What happens? 
     #include <stdio.h>
 // Work an example of this in class (pg. 47)
 void squeeze(char s[], int c) {
     int i, j;
     for (i = j = 0; s[i] != '\0'; i++)
         if (s[i] != c)
         s[j++] = s[i];
     s[j] = '\0';
 }

 int main() {
     char str[20] = "hello there!";
     char c = 'e';
     squeeze(str, c);
     printf("%s\n", str);
 }
